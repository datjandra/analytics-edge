# Analytics Edge Repository

A comprehensive collection of R scripts, datasets, and projects from the [MIT Analytics Edge (15.071x)](https://www.edx.org/course/analytics-edge-mitx-15-071x-2) course on edX. This repository serves as a learning resource for mastering data analytics, statistical modeling, and business applications of analytics.

## 📚 Course Overview

The Analytics Edge course teaches practical applications of analytics in business contexts, combining statistical methods, data visualization, and strategic decision-making. This repository documents the complete learning journey through seven units, each covering different analytical techniques and real-world case studies.

## 📁 Repository Structure

### `/scripts` - R Scripts by Unit
Contains the primary R scripts organized by learning unit, including both instructor-provided code and recitation materials:

- **Unit 1: Introduction to R** - Foundational R programming concepts and syntax
  - `Unit1_IntroductionR.R` - Basic R fundamentals
  - `Unit1_Recitation.R` - Recitation exercises with practical examples

- **Unit 2: Linear Regression & Prediction**
  - `Unit2_Moneyball.R` - Baseball analytics case study using linear regression
  - `Unit2_WineRegression.R` - Wine quality prediction with regression models
  - `Unit2_Recitation.R` - Regression technique practice problems

- **Unit 3: Logistic Regression & Classification**
  - `Unit3_Framingham.R` - Heart disease prediction using Framingham Heart Study data
  - `Unit3_ModelingExpert.R` - Advanced classification modeling techniques
  - `Unit3_Recitation.R` - Classification method exercises

- **Unit 4: Trees & Text Analytics**
  - `Unit4_SupremeCourt.R` - Predicting Supreme Court decisions using CART (Classification and Regression Trees)
  - `Unit4_D2Hawkeye.R` - Text analytics and document clustering application
  - `Unit4_Recitation.R` - Tree-based and text analysis practice

- **Unit 5: Text Analytics & NLP**
  - `Unit5_Twitter.R` - Sentiment analysis on Twitter data using text mining
  - `Unit5_Recitation.R` - Advanced text processing and NLP techniques

- **Unit 6: Clustering & Visualization**
  - `Unit6_Netflix.R` - Movie recommendation system using clustering analysis
  - `Unit6_Recitation.R` - Clustering algorithms and unsupervised learning

- **Unit 7: Visualization & Advanced Topics**
  - `Unit7_Crime.R` - Chicago crime data analysis and geospatial visualization
